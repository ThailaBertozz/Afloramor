# Afloramor – Site de Doações

## Sobre o Projeto
O **Afloramor** é um site criado para facilitar a organização e distribuição de doações em situações de emergência, como desastres naturais.  
A ideia nasceu após as enchentes que atingiram a cidade de **Petrópolis (RJ)** em **15 de fevereiro de 2022**, quando:
- Foram registrados **260 mm de chuva em apenas 6 horas**, acima da média mensal de 232 mm.
- Houve **775 deslizamentos de terra**, centenas de ruas alagadas e destruição de imóveis.
- A tragédia deixou **241 mortos** e prejuízos estimados em **R$ 665 milhões** (cerca de 2% do PIB local).

Durante a tragédia, presenciamos a **falta de estrutura para organizar as doações**, o que resultou em roupas e mantimentos molhados, além de muita desorganização no Alto da Serra.  
Nosso objetivo é que o site ajude a evitar esse tipo de problema em futuras emergências.

## Funcionalidades
- Página inicial com botão **"Doe Agora"**.
- Seção **"Como Doar"** com instruções de doação.
- Página **"Sobre Nós"** com a motivação e apresentação da equipe.
- Sistema de **cadastro e login** para doadores.
- Integração futura com banco de dados para controle de doações.

## Tecnologias Utilizadas
- **HTML5** e **CSS3**: para estrutura e estilização das páginas.
- **Canva**: design de elementos gráficos.
- **Neon Tech**: planejado para o gerenciamento do banco de dados.
